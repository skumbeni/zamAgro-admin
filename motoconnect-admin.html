ℹ️ How It Works

This admin panel shares the same Firebase Realtime Database as the main MotoConnect app. Paste the same firebaseConfig from your Firebase console → Project Settings → Web App.

Once connected, all subscription data syncs in real-time across both panels. Approvals, renewals, and new riders appear instantly on the main app.
